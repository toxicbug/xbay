人面桃花相映红

发表于 2017-03-18 | 分类于 摄影作品 ， 美女 |

倚春而立，裙袂飘飘

诗意的红，点燃一树春光

艳如霞，美如画

是谁家的小女子 

这么斗胆

喊醉三千里麦浪 

桃花庵遇仙记

那时那日此门中， 桃花树下初相逢。
只见仙人种桃树， 未闻仙人看花红。

朝朝期待仙人顾， 日日桃花笑春风。
忽闻仙踪一朝至， 桃花人面分外红。

桃花谷里桃花仙， 桃花美人树下眠。
花魂酿就桃花酒， 君识花香皆有缘。

美酒消愁愁不见， 醉卧花下枕安然。
花中不知日月短， 岂料世上已千年。

不入浊世凡尘染， 情愿枝头做花仙。
春来三月香风送， 便是花奴问君安。

春来了
一棵桃树
层层打开
如水的柔情
缓缓露出
灼灼的心
每个花瓣上
弥漫着
感恩的芬芳
于 是
关于生命
关于美和爱
都有了
具体的
形态和味道
朋友啊
如果我是桃花
你就是那
无边的春光

璀璨夺目，遐想联翩

这本诱人的诗笺

是我用整个青春垒砌的梦

时光如水，多少记忆 

大江东去

唯有你，粉粉的红

依旧灿烂在我的枝头

桃花缭绕着楼阁亭台
再次把妖娆嫁于世间
　　
淡红色的情丝
从花蕊里溢出
羞涩了月光的斑斓
　　
此时
我矜持的灵魂，伏在
一朵半开半闭的花瓣上
期待一张熟悉的笑脸
踏破这一宁静的山间

# 桃花 # 美女
待我长发及腰
好了歌
